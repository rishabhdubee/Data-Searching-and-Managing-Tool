**#Project Title** :
Data Searching & Managing Tool

**#Description** :
This project is made to turn my first year C.S learings into real life working project
the Data Searching & Managing Tool saves the data given by the user and allows the user to search data too

**#Features** :
The user can add multiple data in format(Name,email,number) in the respective entry boxes 
user can also search data entered by different user's

**#Technologies used** :

Language used : **Python**
also : Tkinter , Regular Expression (Regex) , File Handling , Data Stuctures etc

**#How to use**: Data Searching & Managing Tool
First of all the user will be asked a question if he/she wants to enter data or not(search data)
    If the user enters "Yes" the Data Searching & Managing Tool will show options and entery boxes to enter data
and save the data in the database(file)
    If user enters "No" the Data Searching & Managing Tool will show options and entry Boxes to search the data entered in the 
database(file), if no data is stored then the Data Searching & Managing Tool will show no data
